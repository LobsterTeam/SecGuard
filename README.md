# :computer: Hackathor 2020 Contest Project
 
This repository includes source code of the **SecGuard** Android application which is developed for [Hackathor 2020 Contest](https://www.hackathor.net/).

## Dependencies 

- `Flutter` - 1.22.4
- `Android SDK` - 29.0.2
